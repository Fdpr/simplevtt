=== NAVIGATION ===

Drag Canvas:
	- Press and hold Control or Space and left Mouse button
	- Use arrow Keys to navigate
Measure Distance:
	- Press and hold m
Measure Square:
	- Press and hold n
Measure Circle:
	- Press and hold c
Measure Cone:
	- Press and hold v
Toggle grid:
	- Press g
Scale Grid:
	- Hold Alt plus Mouse wheel
	- Hold Alt plus + or -

=== TOKENS ===

Add enemy:
	- Press 1 through 9 to place an enemy token at the cursor
Add hero:
	- Hold shift and Press 1 through 9
Add token from file:
	- Press l
	- Drag and drop (or paste using ctrl plus v) image and select token
Rotate token:
	- Press r or f (hold Shift to snap to 90 degrees)
Scale token:
	- Press + or - (hold Shift to snap to grid size)
Kill token (will be tinted):
	- Press k
Delete token:
	- Press delete or backspace

=== BACKGROUNDS ===

Add background image:
	- Press b
	- Drag and drop (or paste using ctrl plus v) image and select background
Drag background image:
	- Press and hold Shift plus left mouse button
Rotate background image:
	- Press f (hold Shift to snap to 90 degrees)
Scale background image
	- Hold Control plus + or -

== CONFIGURATIONS ==

Load config file or tabletop:
	- Press o
Save current tabletop:
	- Press s

- Custom configurations can be created. See the folder dir/resources/app/example-Config for an example setup.
- Every config from the global config file dir/resources/app/config can be replaced in a local .svttc config file.
- Resources can be placed into the dir/resources/app/res folder for quick access.
